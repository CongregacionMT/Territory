# Territory

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 14.1.3

## Steps to compile

1 - $ git clone https://github.com/Territories-TJ/territory.git

2 - $ cd territory

3 - $ npm install

4 - $ ng serve

5 - Go to http://localhost:4200 :D
